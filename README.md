# Vehicle Parking Management System

A full-stack web application developed using Flask and SQLite for managing vehicle parking reservations, parking lots, and user activities through interactive dashboards and secure authentication.

## Features

### User Features
- User registration and login
- Reserve and vacate parking spots
- View reservation history
- Profile management
- Parking usage summary dashboards

### Admin Features
- Add, edit, and delete parking lots
- Manage parking spots and users
- View reservation details
- Revenue and utilization analytics dashboards
- Search and filter users and parking lots
